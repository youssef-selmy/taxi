# Theme Generation

Generate complete BetterTheme color systems using AI. Choose between **Color-Based Mode** (provide your brand colors) or **Vibe-Based Mode** (describe the feeling you want).

---

## Two Ways to Create Themes

| Mode | Input | Best For |
|------|-------|----------|
| **Color-Based** | Hex colors (e.g., `#1a365d`) | Established brands with defined colors |
| **Vibe-Based** | Descriptive words (e.g., "calm and professional") | New projects exploring visual identity |

---

## Color-Based Mode

Provide your brand colors and let AI derive the complete color system.

### Basic Usage

> "My brand uses navy blue (#1a365d) and gold (#d69e2e). Generate a theme for me."

### What You Get

AI generates ready-to-use Dart code for both light and dark themes:

```dart
// Generated by BetterSuite MCP Server
// Theme: MyBrand

final myBrandLightColorScheme = ColorSystem(
  // Primary colors
  primary: Color(0xFF1A365D),
  primaryBold: Color(0xFF0F2341),
  primaryLight: Color(0xFF2D4A7C),
  primaryContainer: Color(0xFFE8EDF4),
  onPrimary: Color(0xFFFFFFFF),
  onPrimaryContainer: Color(0xFF1A365D),

  // Secondary colors
  secondary: Color(0xFFD69E2E),
  secondaryBold: Color(0xFFB8861A),
  secondaryLight: Color(0xFFE4B84D),
  secondaryContainer: Color(0xFFFDF6E3),
  onSecondary: Color(0xFF1A1A1A),
  onSecondaryContainer: Color(0xFF8B6914),

  // Surface colors
  surface: Color(0xFFFFFFFF),
  surfaceVariant: Color(0xFFF5F7FA),
  onSurface: Color(0xFF1A1A1A),
  onSurfaceVariant: Color(0xFF4A5568),

  // Background
  background: Color(0xFFF7FAFC),
  onBackground: Color(0xFF1A1A1A),

  // ... 90+ additional color properties
  brightness: Brightness.light,
);

final myBrandDarkColorScheme = ColorSystem(
  // Dark mode variant with inverted logic
  primary: Color(0xFF4A7AB8),
  primaryBold: Color(0xFF6B9AD8),
  primaryLight: Color(0xFF2D5A8A),
  primaryContainer: Color(0xFF1A365D),
  onPrimary: Color(0xFFFFFFFF),
  onPrimaryContainer: Color(0xFFB8D4F0),

  // ... dark mode variants
  brightness: Brightness.dark,
);
```

### Input Options

| Colors | Description |
|--------|-------------|
| **Primary only** | AI derives secondary and accent colors |
| **Primary + Secondary** | You control both main colors |
| **Primary + Secondary + Tertiary** | Full control over the palette |

### Examples

**Single primary color:**
> "Generate a theme with primary color #6366F1 (indigo)"

**Primary and secondary:**
> "My brand uses #10B981 (emerald) as primary and #F59E0B (amber) as secondary"

**Full palette:**
> "Create a theme with primary #8B5CF6, secondary #EC4899, and tertiary #14B8A6"

## Vibe-Based Mode

Describe the feeling you want, and AI intelligently generates a complete color palette.

### How It Works

Instead of hex codes, describe the mood, personality, or aesthetic:

> "Create a calm and professional theme for a financial app"

AI understands design psychology and generates colors that match your description.

### Vibe Examples

#### Calm & Professional

> "Generate a theme that feels calm, trustworthy, and professional"

```dart
// AI generates muted blues, soft grays, subtle accents
final calmProfessionalLight = ColorSystem(
  primary: Color(0xFF3B82F6),      // Trustworthy blue
  secondary: Color(0xFF64748B),    // Professional slate
  surface: Color(0xFFFAFAFA),      // Clean white
  // Muted, balanced color palette
);
```

#### Energetic & Playful

> "I want an energetic, playful theme for a fitness app"

```dart
// AI generates vibrant, high-contrast colors
final energeticPlayfulLight = ColorSystem(
  primary: Color(0xFFEF4444),      // Energetic red
  secondary: Color(0xFFF97316),    // Vibrant orange
  surface: Color(0xFFFFFFFF),      // Crisp white
  // Bold, dynamic color palette
);
```

#### Luxurious & Minimal

> "Create a luxurious, minimal theme for a high-end product"

```dart
// AI generates sophisticated, restrained colors
final luxuryMinimalLight = ColorSystem(
  primary: Color(0xFF1F2937),      // Elegant charcoal
  secondary: Color(0xFFD4AF37),    // Gold accent
  surface: Color(0xFFFFFBF5),      // Warm white
  // Refined, understated palette
);
```

#### Eco-Friendly & Natural

> "Generate a nature-inspired theme for a sustainability app"

```dart
// AI generates organic, earthy colors
final ecoNaturalLight = ColorSystem(
  primary: Color(0xFF059669),      // Forest green
  secondary: Color(0xFF92400E),    // Earth brown
  surface: Color(0xFFFFFDF7),      // Natural white
  // Organic, grounded palette
);
```

### Vibe Keywords

Use these descriptive words to guide the AI:

| Category | Keywords |
|----------|----------|
| **Mood** | calm, energetic, serious, playful, warm, cool |
| **Style** | minimal, bold, elegant, modern, classic, futuristic |
| **Industry** | corporate, creative, medical, tech, luxury, casual |
| **Feeling** | trustworthy, exciting, soothing, powerful, friendly |

### Combining Vibes

Be specific by combining multiple descriptors:

> "Create a theme that's modern and tech-forward but also warm and approachable"

> "I need a professional theme that's not boring - add some creative energy"

> "Generate a calming healthcare theme with a touch of optimism"

## Using Generated Themes

### Apply to Your App

```dart
import 'package:better_design_system/better_design_system.dart';

// Use your generated color systems
MaterialApp(
  theme: BetterTheme.fromColorSystem(
    myBrandLightColorScheme,
    context.isDesktop,
  ),
  darkTheme: BetterTheme.fromColorSystem(
    myBrandDarkColorScheme,
    context.isDesktop,
  ),
  themeMode: ThemeMode.system,
)
```

### Access Colors in Widgets

```dart
final colors = Theme.of(context).extension<ColorSystem>()!;

Container(
  color: colors.primary,
  child: Text(
    'Hello',
    style: TextStyle(color: colors.onPrimary),
  ),
)
```

## Tips for Best Results

### Be Specific About Use Case

> "Generate a theme for a banking app that needs to feel secure and trustworthy"

### Mention Accessibility

> "Create a high-contrast theme for better readability"

### Reference Existing Brands

> "I want something similar to Stripe's visual style but with green as the primary color"

### Iterate

> "That's close, but can you make the primary color slightly warmer?"
